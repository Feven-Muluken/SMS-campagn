# Backend setup

## Password reset email requirements

The reset link is generated by the API, but it is only sent to user email when SMTP is configured.

Create a `.env` file in `backend` with:

```env
PORT=5000
JWT_SECRET=your_jwt_secret
FRONTEND_URL=http://localhost:5173

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your_email@gmail.com
SMTP_PASS=your_app_password
SMTP_FROM="Afroel SMS <your_email@gmail.com>"
```

## Notes for Gmail

- Use a Gmail App Password (not your normal Gmail account password).
- Enable 2-Step Verification, then generate an App Password.
- Keep `SMTP_PORT=587` and `SMTP_SECURE=false` for STARTTLS.

## Run

```bash
npm install
npm run dev
```

After this, `POST /auth/forgot-password` sends the reset link to the target email.
